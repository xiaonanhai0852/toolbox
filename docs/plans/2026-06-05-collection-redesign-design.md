# 收藏夹重构设计文档

**日期**: 2026-06-05
**状态**: 已批准
**方案**: 重新设计侧边栏

## 1. 整体架构

### 当前结构
```
Sidebar (文件夹树 + 笔记列表 + 搜索 + 排序 + 分页)
```

### 重构后结构
```
├── FolderPanel (文件夹树 + 搜索 + 新建文件夹)
└── NotePanel (笔记列表 + 排序 + 分页 + 批量操作)
```

### 设计目标
- UI/UX 改进：展开/折叠动画、响应式布局、拖拽体验优化
- 功能增强：文件夹内搜索、笔记数量显示、批量操作

---

## 2. FolderPanel 组件

### 功能
- 文件夹树显示（平级，无嵌套）
- 文件夹内搜索（实时过滤）
- 新建文件夹按钮
- 文件夹重命名、删除（hover 显示操作按钮）
- 拖拽目标区域（视觉反馈）
- 显示每个文件夹的笔记数量

### 交互细节
- 点击文件夹 → 高亮选中，NotePanel 更新
- 点击"全部笔记" → 显示所有笔记
- 拖拽笔记到文件夹 → 显示拖拽高亮，释放后移动
- 文件夹名称过长 → 省略号 + tooltip

### 样式
- 使用 CSS 变量保持一致性
- 文件夹项高度：36px
- 图标 + 名称 + 数量 badge
- hover/selected 状态过渡动画

---

## 3. NotePanel 组件

### 功能
- 笔记列表显示
- 排序选择器（最后修改、创建时间）
- 分页控件
- 批量操作模式（复选框 + 批量移动/删除）
- 拖拽笔记到文件夹

### 交互细节
- 笔记项显示：标题 + 日期 + 文件夹标签（可选）
- 拖拽笔记 → 显示拖拽预览，文件夹高亮
- 批量模式 → 顶部显示操作栏（移动到... / 删除）
- 分页 → 上一页/下一页 + 页码显示

### 样式
- 笔记项高度：48px
- hover/selected 状态
- 拖拽时半透明效果
- 批量模式复选框样式

---

## 4. 响应式布局

### 断点设计
- 桌面（≥1024px）：双栏布局，FolderPanel + NotePanel 并排
- 平板（768px-1023px）：可折叠侧边栏，点击切换
- 移动端（<768px）：全屏切换，底部导航栏切换文件夹/笔记

### 移动端交互
- 底部导航栏：文件夹图标 | 笔记图标
- 点击文件夹 → 切换到笔记列表
- 返回按钮 → 回到文件夹列表
- 拖拽 → 长按触发，底部显示目标文件夹

### 动画效果
- 侧边栏展开/折叠：200ms ease-out
- 文件夹展开/折叠：150ms ease-out
- 拖拽高亮：100ms ease-in
- 页面切换：slide 动画

---

## 5. 拖拽体验优化

### 拖拽状态
1. 开始拖拽 → 笔记项半透明 + 拖拽预览
2. 拖拽到文件夹 → 文件夹高亮（边框 + 背景色变化）
3. 释放 → 移动笔记，显示成功提示
4. 拖拽到"全部笔记" → 移出文件夹

### 视觉反馈
- 拖拽预览：笔记标题 + 半透明背景
- 文件夹高亮：2px primary-color 边框 + 背景色变化
- 拖拽过程中禁止文本选择
- 拖拽结束后恢复原状态

### 动画效果
- 拖拽开始：100ms ease-in
- 文件夹高亮：50ms ease-out
- 移动完成：200ms ease-out（笔记飞入文件夹动画）
- 成功提示：fadeIn 200ms

---

## 6. 批量操作

### 批量模式
- 触发：点击"批量操作"按钮 或 长按笔记项
- 退出：点击"取消" 或 完成操作后自动退出

### 批量操作功能
- 复选框选择笔记
- 全选/取消全选
- 批量移动到文件夹（下拉选择目标文件夹）
- 批量删除（确认对话框）

### UI 变化
- 进入批量模式 → 顶部显示操作栏
- 操作栏：全选 | 已选 N 项 | 移动到... | 删除 | 取消
- 笔记项左侧显示复选框
- 选中笔记背景色变化

### 交互细节
- 移动到... → 下拉菜单显示文件夹列表
- 删除 → 确认对话框，显示删除数量
- 操作完成后 → 退出批量模式，刷新列表

## 7. 技术实现

### 组件结构
```
src/features/notes/
├── components/
│   ├── FolderPanel.jsx
│   ├── NotePanel.jsx
│   ├── FolderItem.jsx
│   ├── NoteItem.jsx
│   └── BatchToolbar.jsx
├── pages/
│   └── NotesLayout.jsx
└── styles/
    └── notes.css
```

### 状态管理
- `selectedFolderId` - 当前选中的文件夹
- `notes` - 笔记列表
- `folders` - 文件夹列表
- `isBatchMode` - 批量模式状态
- `selectedNoteIds` - 选中的笔记 ID 集合
- `isMobileView` - 移动端视图状态

### API 调用
- `GET /api/folders` - 获取文件夹列表
- `GET /api/notes?folder_id=X` - 获取文件夹内的笔记
- `POST /api/folders` - 创建文件夹
- `PUT /api/folders/:id` - 更新文件夹
- `DELETE /api/folders/:id` - 删除文件夹
- `PUT /api/notes/:id` - 移动笔记到文件夹
- `POST /api/notes/batch-move` - 批量移动笔记
- `POST /api/notes/batch-delete` - 批量删除笔记
